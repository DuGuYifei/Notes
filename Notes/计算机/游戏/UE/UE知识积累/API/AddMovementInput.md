# APawn::AddMovementInput

沿给定的世界方向矢量（通常归一化）添加运动输入，按“ScaleValue”缩放。

| 参数           | 描述                                                                                |
| -------------- | ----------------------------------------------------------------------------------- |
| WorldDirection | 世界空间中应用输入的方向 input                                                      |
| ScaleValue     | 缩放以应用于输入。这可用于模拟输入，即值 0.5 应用正常值的一半，而 -1.0 将反转方向。 |
| bForce         | 如果为 true，则始终添加输入，忽略 IsMoveInputIgnored（） 的结果。                   |
