# -p

`--publish` 用于开放端口，作用类似于防火墙

`-p :80` 会默认给一个

-p指令有哪些用法？
大家用的最多的可能就是-p 80:80事实上，还有其他用法：

`-p 8080:80`

将容器的TCP80端口映射到宿主机的8080端口

`-p 192.168.1.100:8080:80`

将容器的TCP80端口映射到IP地址为192.168.1.100宿主机的8080端口

`-p 8080:80/udp`

将容器的UPD80端口映射到宿主机的8080端口

`-p 8080:80/tcp -p 8080:80/udp`

同时映射TCP和UDP端口
