1. C++ 项目 -> 属性 -> vc++目录
2. 包含目录 -> 新增
   ![](2022-07-24-11-30-58.png)
3. 库目录 -> 新增
   ![](2022-07-24-11-31-44.png)
4. 链接器 -> 输入 -> 附加依赖项 -> 编辑
   ![](2022-07-24-11-32-32.png)
