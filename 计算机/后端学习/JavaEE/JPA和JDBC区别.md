# JPA和JDBC区别

JPA是比JDBC高一层的抽象，JPA本质上时使用JDBC的，JDBC更接近数据库，使用sql之类的。JPA还是属于应用层面的API。

- Java Database Connectivity (JDBC) – low level of abstraction;
- Java Persistence API (JPA) – high level of abstraction;
