# 图元片元光栅化

1. 顶点（vertex）：图元（如线段、三角形、圆等几何图形）由顶点+边组成，由用户及其建立的模型确定.
2. 图元（primitive）：描述对象的几何要素的输出图元，称为几何图元，简称图元. 如点、直线段、圆、二次曲线、曲面等. 只包含每个顶点和顶点的数据。
3. 片元（fragment）：光栅化过程的产物，光栅化将一个图元转变成二维图像，每个图像点都包含颜色、深度、纹理等数据，将该点和相关信息称为一个片元。有时候一个像素被多个三角形覆盖，意味着会创建多个片元。注意，这里的多个三角形覆盖不是说采样时的那种被两个三角形分开覆盖，而是z-buffer下前后不同的三角形都完全覆盖。
4. 像素（pixel）：最终呈现在屏幕上的点，包含RGBA值的图像最小单元.

生成顺序：顶点 -> 图元 -> 片元 -> 像素
