# 基础运算符，数学

## `+` `-` `*` `/` `%` `**`

`**`求幂

```
"my" + "string" = "mystring"
'1' + 2 = "12"
2 + '1' = "21"
2 + 2 + '1' = "41"
'1' + 2 + 2 = "122"
6 - '2' = 4
'6'/'2' = 3
```

```
let a = "2"
let b = "3"
alert(a + b); //"23"
alert(+a + +b); //5
alert(Number(a) + Number(b)); //5
```
