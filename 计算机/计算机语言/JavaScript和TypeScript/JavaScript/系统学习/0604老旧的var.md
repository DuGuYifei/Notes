# 老旧的var

本文用于理解旧脚本

[变量](0204变量.md)章节里三种变量声明方式：
1. let
2. const
3. var

`var`声明和`let`相似。大部分情况下可以互相代替。

但是：

## var没有块级作用域
```javascript
if (true) {
  var test = true; // 使用 "var" 而不是 "let"
}

alert(test); // true，变量在 if 结束后仍存在
```

```javascript
if (true) {
  let test = true; // 使用 "let"
}

alert(test); // ReferenceError: test is not defined
```
